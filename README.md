<h1>Mailer en AJAX</h1> 

lien: https://mailthp.herokuapp.com/



<h2>Utilisation en local</h2>


> <p>git clone https://github.com/nicoc-irtanos/email_viewer_ajax_thp.git

> cd email_viewer_ajax_thp

> bundle install 

> rails db:create

> rails db:migrate

> rails db:seed

> rails server  

Go sur http://localhost:3000/
